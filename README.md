# MoodMate - AI Chatbot 💬

MoodMate is an AI-powered chatbot designed to help users track their mood and interact conversationally.

## Features
- AI-based conversation
- Mood tracking
- Data visualization
- User-friendly interface

## Tech Stack
- Python
- Streamlit
- NLP
- Data Visualization

## Future Improvements
- Advanced sentiment analysis
- Personalized recommendations
