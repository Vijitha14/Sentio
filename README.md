# 💙 Sentio – AI Emotion Analysis Platform

Sentio is an AI-powered emotion analysis platform that analyzes user text to identify emotions and sentiment using Natural Language Processing (NLP). The application provides meaningful insights into emotional tone through an intuitive and interactive interface.

---

## ✨ Features

- 😊 Emotion Detection
- 💬 Sentiment Analysis
- 🧠 AI-powered Text Processing
- 📊 Visual Emotion Insights
- ⚡ Fast and Interactive Interface
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- Python
- NLP
- Machine Learning
- Streamlit / Flask
- HTML
- CSS
- JavaScript

---

## 🚀 How It Works

1. Enter text.
2. The AI processes the input.
3. Emotion and sentiment are analyzed.
4. Results are displayed through an intuitive interface.
